## 事業をエンジニアリングしよう提案編の回答は以下に記述してください
```
・選択した事業側の課題
もっと女性が使いやすいようなサービス設計にする必要があるのではないか？

・提案内容
女性限定のもくもく会を実施できるようにする
→男性が多いため、女性が参加したくても『行きづらい』『不安』と感じ、参加に踏み出せていない
ユーザーがいることが考えられる。女性同士で繋がれる仕組みを作る。


・実装方針
ユーザー登録時に性別の項目を追加する
女性ユーザーはもくもく会作成時に女性限定かどうかを選べるようにする
男性ユーザーはもくもく会作成時に女性限定を選べない
イベント一覧画面で対象のイベントについては”女性限定”と明記する
男性ユーザーは女性限定イベントの”このもくもく会に参加する”ボタンを押せない
```
